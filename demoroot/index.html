Welcome to And/Or

This is a demo of And/Or1, a extremely light weight JSON object web service. The dataset collection is "demo-colloction-01.ds". HTML output is base on static files. The UI is located in the document root under /demo-collection-01/.

  • Web UI Landing page - should trigger a Basic AUTH loging
  • JSON views (raw And/Or access)
    • List of Keys - a JSON list of object keys
    • Single Object - a JSON view of read object id "100".
    • List of Objects - An ordered array of objects with keys 100, 101, 102 and 103
  • HTML views (static pages using JavaScript to format raw output)
    • List objects keys
    • Object - an HTML view of object id "100".
    • Objects - an HTML view of object id "100", "101", "102", and "103".

Footnotes

  1. And/Or is named after a character in Ruby stories produced by ZBS
