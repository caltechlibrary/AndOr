(Example) documentation for AndOr

The following are actions available from the command line usage of AndOr. While it takes avantage of the rich library of packages and tools developed at Caltech Library AndOr itself is a self contained command line program.

  • init - intialize dataset collections for use by AndOr
  • add-workflow - add a workflow
  • list-workflows - list workflows
  • remove-workflow - remove a workflow
  • add-user - add a user
  • add-user-workflow - add a user to a workflow
  • remove-user-workflow - remove a workflow for a user
  • list-users - list users and assigned workflows
  • remove-user - remove a user
  • config - configure AndOr web service
  • start - start AndOr web service
  • restart - restart AndOr web service
  • stop - stop AndOr web service
